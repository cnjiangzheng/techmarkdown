###打包Linux 执行文件
1、打开到执行go文件目录位置
2、set GOOS=linux
3、go build xxx.go

###打包win执行文件
1、打开到执行go文件目录位置
2、set GOOS=windows
3、go build xxx.go

###打包mac执行文件
1、打开到执行go文件目录位置
2、set GOOS=darwin
3、go build xxx.go

###查看go环境变量
go env

###启动脚本
start.sh
```bash
#!/bin/bash
cd `dirname $0`
cd ../
mkdir -p log
ps -aux |egrep "EHH_v_0_1_U0" | cut -c 9-15 | xargs kill -9 >/dev/null 2>&1
nohup ./bin/EHH_v_0_1_U0 -debug >> log/console_output.log 2>&1 &
```
###停止脚本
stop.sh
```bash
#!/bin/bash
ps -aux |egrep "EHH_v_0_1_U0" | cut -c 9-15 | xargs kill -9 >/dev/null 2>&1
```
